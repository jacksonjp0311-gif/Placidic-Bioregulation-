# Codex Delta Phi - PBSA v2.1 Routed-Suite Validation

## Status

Canonical PBSA v2.1 routed-suite validation layer.

## Purpose

PBSA v2.1 validates whether regime-routed PBSA improves or preserves behavior compared with champion-only, baseline-only, candidate-only, and reject/manual-review controls.

## Core invariant

Route selection is not enough. A routed policy must be tested against non-routed controls.

## Added surfaces

- validation policy config
- control policy definitions
- route advantage metrics
- route preservation score
- route failure surface
- routed validation report

## Lock

Routed validation is not biological validation. Routed advantage is not medical evidence. Validation is not automatic kernel replacement.
