# Codex Delta Phi - PBSA v3.0 Public Research Package

## Status

Canonical PBSA v3.0 public research package layer.

## Purpose

PBSA v3.0 freezes the release-candidate-ready PBSA system into a public computational research package with publication abstract, evidence summary, reproducibility commands, public limitations, claim-boundary locks, release tag metadata, and public package report.

## Core invariant

Public release is not biological validation. Public audit readiness is not medical safety. Release readiness is not mechanism proof.

## Added surfaces

- public package policy
- public release manifest
- publication abstract
- evidence summary
- public limitations
- public command surface
- public claim-boundary table
- public package readiness verifier
- public package report
- release tag preparation metadata

## Lock

Public research package status is computational audit readiness only. It is not biological validation, medical validation, clinical safety, or physiological mechanism proof.
